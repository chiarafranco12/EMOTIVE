Emotive: face and emotion recognition

Emotive is a program able to recognize your face and your expression. You can use it using this website, or you can download the source code, for develop your own applications

Python

The backend application is written in Python we decided to use Python because is the best for Machine Learning programs

TensorFlow

We used TensorFlow for the whole Machine Learning part. We downloaded a dataset of images called FER 2013, from Google

Tailwind CSS

The front end web application is written using Tailwind CSS, in order to make a modern and responsive website

Flask

Flask is a web framework written in Python. We used it to use the backend program
